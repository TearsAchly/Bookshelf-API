When building the Bookshelf API, you'll naturally need to test to ensure that the API meets the specified criteria. We've provided Postman Collection and Environment files for testing purposes. Please download the files from the following link:

[Postman Bookshelf API Test Collection and Environment](https://github.com/TearsAchly/Bookshelf-API)

You'll need to import both files into Postman to use them. Here's how:

Extract the downloaded files to produce two JSON files.

In the Postman application, click the import button located above the left panel.

   ![Import button](https://raw.githubusercontent.com/TearsAchly/Bookshelf-API/master/assets/4.png)

Then, click the "Upload Files" button to import the two JSON files extracted earlier.

   ![Upload Files button](https://raw.githubusercontent.com/TearsAchly/Bookshelf-API/master/assets/1.jpeg)

</br>After that, the Bookshelf API Test Collection and Environment will be available in your Postman.
